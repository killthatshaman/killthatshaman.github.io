在另一台电脑上折腾了一下一直用的网易云UWP版的安装细节.

安装

从这里下载uwp安装包，然后按照说明安装。
如果连接不上microsoft store记得关代理。

登录

参考这里

在开了系统代理的情况下显示网络错误的解决办法

参考这里,找到网易云uwp的SID即可.

更快捷的方法,我使用的NetLimiterPro自带显示SID,无需打开regedit.
